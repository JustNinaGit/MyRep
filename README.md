\# Моє Репо



Навчальний проєкт для вивчення Git та GitHub.



У межах проєкту буде виконано:

\- створення локального Git-репозиторію;

\- робота з комітами;

\- робота з гілками;

\- використання GitHub;

\- робота з файлами .gitignore;

\- використання команд push, pull та clone;

\- створення Pull Request.
