\# Моє Репо



Навчальний проєкт для вивчення Git та GitHub.



У межах проєкту буде виконано:

\- створення локального Git-репозиторію;

\- робота з комітами;

\- робота з гілками;

\- використання GitHub;

\- робота з файлами .gitignore;

\- використання команд push, pull та clone;
## Висновок

У цьому проєкті було створено локальний Git-репозиторій, виконано кілька комітів, підключено GitHub та відправлено зміни у віддалений репозиторій.
\- створення Pull Request.
